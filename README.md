## 💼 Featured Project

<div align="center">

### 🔍 Russian Business INN Lookup Tool

</div>

> **Real business automation tool** — processes 27,000+ company tax numbers (INN) through the DaData API and exports structured data to Excel.

**What it does:**
- 📥 Reads INN numbers from CSV / Excel input files
- 🌐 Queries [DaData API](https://dadata.ru) for each company — name, director, address, OKVED, legal status
- 🔄 Auto-retries failed requests, resumes from checkpoint (no wasted API quota)
- 💾 Saves progress every 100 records — crash-safe
- 📊 Exports everything to a clean `.xlsx` table (27,000+ rows)

**Tech used:** `Python 3.14` · `requests` · `openpyxl` · `DaData REST API`

**Scale:** Processed **27,053 companies** across 3 days (free API tier, 10k/day limit)

```
✅ Active companies:   ~19,589
🔴 Liquidated:         ~2,468
🟠 Bankruptcy:            711
🟡 Being liquidated:      200
```
